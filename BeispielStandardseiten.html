Standardseiten

Auf Standardseiten finden User die gesuchten Informationen. Welche Elemente Sie für die Standardseiten der RUB nutzen können, lesen Sie hier.

Die Standardseite ist in der Desktopansicht dreispaltig, wobei die linke Seitenspalte für das Menü und gegebenenfalls maximal ein Abteilungslogo vorgesehen ist, die rechte Seitenspalte für Zusatzinformationen.

Aufbau einer Standardseite

Die ersten Elemente auf jeder Standardseite sind

  1. ein Bild (im Format 999 x 666 Pixel)
  2. die Seitenüberschrift
  3. eine einordnende Dachzeile (optional; einheitliche Nutzung im jeweiligen Bereich)
  4. ein kurzer Einleitungstext
  5. weitere, frei zu wählende Elemente wie sie auf dieser Seite beschrieben sind
  6. abschließend eine Verweisfläche, die die User auf die jeweils nächsthöhere Ebene führt

Die Webseite auf mobilen Endgeräten

Beim Aufbau der Seite wird stets bedacht, dass diese für mobile Endgeräte (Handys, Tablets) anders ausgegeben wird als auf einem Desktop: Die mobile Ansicht ist einspaltig, die drei Spalten der Desktopansicht bauen sich von links nach rechts untereinander auf, also

  1. wenn vorhanden – Titel und Logo aus der linken Spalte,
  2. der Seiteninhalt, bei Elementen wie Teasern oder Verweisflächen untereinander von rechts nach links
  3. die Inhalte der rechten Spalte
  4. der Fußbereich der Seite – erst der Instanz-Fußbereich, dann der allgemeine Fußbereich der RUB-Seiten

Darstellung von Text und Inhalt

Die Textspalte einer Standardseite ist auf optimale Lesbarkeit ausgelegt. In der Regel „scannen“ die User die Seiten nach den gesuchten Informationen ab und wollen schnell erfassen, ob sie sich auf der richtigen Seite befinden. Daher müssen die Inhalte gut ausgezeichnet sein.

Inhalte gliedern

Folgende Darstellungsmittel stehen zur Verfügung, um den Text aufzulockern:

  • Aufzählungen, nummeriert oder unnummeriert
  • Absätze (weit mehr, als gewöhnlich in Texten genutzt werden)
  • Bilder, Infografiken, Bildergalerien oder Videos (mehr zu Medien in RUB-Web)
  • Zwischenüberschriften
  • Zitate
  • Verweisflächen und verlinkte Buttons
  • Infoboxen

Ist ein Text durch diese Darstellungsformen strukturiert aufgebrochen, stellen User schnell fest, ob die Seite die gesuchten Informationen enthält – selbst wenn diese ein wenig länger ist.

Überschriften

Zwischenüberschriften gliedern den Text und wirken sich auch auf Suchmaschinen aus. Daher tauchen die wichtigsten Schlagworte des behandelten Themas in Zwischenüberschriften auf. Überschriften werden in der angedachten Struktur genutzt. Es stehen vier Überschriftentypen (Überschrift 1 bis Überschrift 4) zur Verfügung: Die Überschrift 1 ist der Überschrift der Seite vorbehalten. Faltboxen

Die Faltbox ist für verschiedene Informationen gedacht. Faltboxen gibt es in verschiedenen Formaten:

Die Infobox

Mit dieser Box werden Zusatzinformationen zum Hauptinhalt geliefert. Dies können zum Beispiel Adressen, Literaturhinweise oder ergänzende Informationsblöcke sein. Bilder können optional eingefügt werden. Das Icon in der linken Ecke kann angepasst werden.

Die Faltbox

Nutzen Sie die Faltbox, um Informationen zu bündeln. Die Faltbox ist kein eigener Inhaltsbereich – sie liefert zusätzliche Informationen zu einem Thema. Sie kann ein- und ausgeblendet werden. Deshalb stehen keine elementaren Informationen in der Faltbox. Für Informationen, die immer sichtbar sind, nutzen Sie die Infobox. Bilder können optional eingefügt werden. Das Icon in der linken Ecke kann angepasst werden.
